## Example

```text
material:DIAMOND,amount:64 ; material:EMERALD,amount:32
```

## Properties

* Material
* Data
* Amount
* Name
* Lore
* Texture
* ModelData

## Note

* Match items with the specified properties, seperate each property by a `,`
* You can specify multiples items by seperating them with `;`
