# Page

## Usage

```text
((set|switch)?(-)?(shape|page))|shape|page
```

## Example

```yaml
- 'page: 0'
```

## Note

* The layouts numbers start from 0, so the layout 0 is the first one, the layout 1 is the second etc...
* You can use placeholders and arguments
