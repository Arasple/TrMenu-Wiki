# Installation

## Pré-Requis

{% hint style="info" %}
Au premier démmarage, TrMenu doit avoir un accès à Internet.
{% endhint %}

| Server Software | Supported? |
| :--- | :--- |
| Spigot | 1.8-1.16 |
| Paper | 1.8-1.16 |
| Akarin | × |
| Arclight \(Forge\) | √ |
| CatServer \(Bukkit+Forge\) | √ |

## Installation

* Mettez le fichier `TrMenu.jar` dans votre dossier plugins
* Redémarrer le serveur
