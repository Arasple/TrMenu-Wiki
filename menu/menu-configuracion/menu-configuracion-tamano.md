# Tamaño

**Formas de escribirlo**

* **Normal:** Size
* **RegEx:** \(size\|row\)\(s\)?

### Tamaño del menú

* En caso de usar un menú normal o tipo CHEST puedes especificar el número de líneas.
* Usar únicamente esto en caso de no haber configurado el diseño del menú.

```yaml
Size: 4
```
