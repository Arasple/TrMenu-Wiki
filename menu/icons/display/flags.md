# Flags

## Usage

```text
flag(s)?
```

## Example

```yaml
flags:
- 'HIDE_ENCHANTS'
- 'HIDE_ATTRIBUTES'
```

## Available Flags

* HIDE\_ENCHANTS
* HIDE\_ATTRIBUTES
* HIDE\_UNBREAKABLE
* HIDE\_POTION\_EFFECTS
* HIDE\_DESTROYS
* HIDE\_PLACED\_ON
