# JavaScript

## Usage

```text
(java)?(-)?script(s)?|js
```

## Examples

```yaml
- 'js: player.setNoDamageTicks(600)'
- 'js: player.setVelocity(player.getVelocity().setY(25))'
- 'js: bukkitServer.broadcastMessage("A Broadcast Message")'
```
